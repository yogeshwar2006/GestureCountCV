# ✋ Vision-Based Finger Counting System

A real-time **computer vision calculator** that performs mathematical operations using **hand gestures detected through a webcam**.

This project detects the number of fingers from both hands and allows users to perform **addition, subtraction, multiplication, and division** by selecting operations on the screen.

---

##  Features

- Real-time hand tracking
- Finger counting using computer vision
- Supports two-hand input (0–10)
- Gesture-based number detection
- Interactive operation buttons (+ − × ÷)
- Real-time calculation results
- Simple visual UI using OpenCV

---

##  Development Stack

- Python
- OpenCV
- MediaPipe
- Computer Vision
- Hand Landmark Detection

---

##  How It Works

1. The webcam captures the user's hand gestures.
2. The system detects hand landmarks using MediaPipe.
3. Finger counting determines the numbers shown by the **left and right hands**.
4. The user selects an operation by touching a button on the screen.
5. The calculator performs the selected operation and displays the result.
